University of Minnesota Campus Interactive Map

Interactive campus map allows you to access campus information’s like bus stops, bus routes, arts venue, sports venue etc. allowing you to explore the campus on your fingertips. It’s a mobile responsive environment, made available to give a user friendly interface for the students, staff and other people affiliated with university. The application is build using web AppBuilder developers edition which uses JavaScript API, html, CSS as its development platform. Working on this project has been a great opportunity for my personal and professional development. This project has given me exposure and opportunity to portray my work on a larger scale.

Simple Desktop Viewer

Simple Desktop Map viewer is developed using ArcGIS Runtime SDK for .Net 10.2.7. It allows you to add, remove, view, navigate through Web Services. The user can add new web services or use the existing one. It also allows identifying features. This project can be used in companies where they don’t want to invest in 100s of license copies for ArcMap but still want some of their employees to have the ability to view the internal services in a very basic viewer. The app can be can customized according to the company’s needs. All you need a developer’s license and one-time investment in application development.

Snow tow Information Map

Simple Data Extractor

This is a Simple python tool which allows you to extract data from a feature class using select by attribute or select by location or by both or none into a new Feature class. Processing time is reduced to 33 seconds from 3 minutes.

University of Minnesota Ethnicity 2013 -2015

This is a Spatial Humanities project, which is an analysis of influence of Ethnicity on choosing education trades, levels. The objective of the analysis is to learn about the diverse population around the campus, to visualize the Ethnicity patterns, to study the pattern respect to the level of studies and departments, to analyze the relationship between the ethnicity and the trade. This analysis do not promote racial discrimination.
